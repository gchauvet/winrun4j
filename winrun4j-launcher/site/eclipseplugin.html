WinRun4J
Configurable Java Launcher for Windows
Eclipse Plugin
The Eclipse Plugin for WinRun4J provides a deep level of integration of the launcher within the Eclipse IDE. It has the following features:
  • Launch any Java Application with WinRun4J (via right-click or launch configuration)
  • Export your Java Application as a single Windows Executable (with necessary jars embedded)
  • Provides improved windows integration from Eclipse (eg. JNI access to Registry).
Note that this plugin is for Windows only.
Download
The Eclipse Plugin for WinRun4J can be downloaded from the eclipse Update Site:

http://winrun4j.sourceforge.net/
